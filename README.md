# 📖 Spinning Grace- A Musical Carousel of God's Love<a name="about-project"></a>

**Spinning Grace- A Musical Carousel of God's Love** 
<br>

Spinning Grace: A Musical Carousel of God's Love. This JavaScript repository employs HTML, CSS, JavaScript, Webpack, Linters, Visual Studio Code, and GitHub Pages to create a carousel featuring three songs that symbolize God's almighty love: Amazing Grace, This I Know, and How Great is Our God.

<br>

## 💻 Getting Started <a name="getting-started"></a>

- Create a local directory that you want to clone the repository.

- Open the command prompt in the created directory.

- On the terminal run this command git clone `https://github.com/annaeliana7/Spinning-Grace--A-Musical-Carousel-of-God-s-Love.git`

- Go to the repository folder using command prompt cd `Spinning Grace- A Musical Carousel of God's Love`

- Install the dev dependencies for linters run npm install.
